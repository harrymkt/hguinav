## 0.11.5 unreleased
- Hugo 0.161.1 is now the minimum supported version.
- All links now use absolute URL.
- Change from `language.Name` and `language.name` (config) to `language.Label` / `language.label`, as it is deprecated.
- `language.LanguageCode` and `language.languageCode` are now deprecated, using `language.Locale` and `language.locale`.
